1 // NODE MANAGEMENT SOP

Establish a clean, isolated Python node for local-first automation. Always use a dot prefix (.venv) to keep the environment hidden from standard file scans and recursive logic loops.

// INITIALIZATION SEQUENCE SHELL 
python3 -m venv .venv
source .venv/bin/activate
which python
pip install playwright pandas
pip freeze > requirements.txt
deactivate

2 // MEDIA ARSENAL EXTRACTION

De-compile public media assets from target domains. This sequence bypasses third-party compression scripts and preserves raw, high-fidelity streams.

// YT-DLP EXTRACTION MANIFEST 
# Download maximum quality video and audio merged
yt-dlp "[TARGET URL]"


# Extract audio stream only (M4A format conversion)
yt-dlp -f 'ba' -x --audio-format m4a "[TARGET URL]"


# Frame-accurate clip extraction with local re-encoding
ffmpeg -ss 00:00:40 -to 00:01:50 -i input.mp4 -c:v libx264 -c:a aac output.mp4

3 // SYSTEM RECONNAISSANCE

Query the local hardware node directly. Avoid the lag of standard system settings wrappers. These commands poll the kernel database immediately to identify partition capacities and purge useless files.

// THE COMPLETE DF (DISK FREE) COMMAND INDEX 
# Human-readable binary partition status (1024-byte blocks)
df -h

# Human-readable decimal partition status (1000-byte blocks)
df -H

# Target only the partition containing the current active directory
df -h .

# Local file systems only (suppresses slow network mounts)
df -lh

# Display inode usage statistics instead of block capacity
df -i

# Include all system-level and empty file systems
df -a

# Filter output to display only APFS formatted volumes
df -t apfs

# Exclude specific file systems from the output
df -x devfs

# Display space allocation in 1-kilobyte blocks
df -k

# Display space allocation in megabytes
df -m

# Display space allocation in gigabytes (macOS specific)
df -g
// DIRECTORY CLEANUP AND DISK UTILITIES 
# Drill-down: List directories sorted by size
du -sh ~/* | sort -h

# Identify which application holds port 3000
lsof -i :3000

# Safe homebrew cleanup to purge previous package versions
brew cleanup

# Purge local hidden Time Machine snapshots
tmutil listlocalsnapshots /
sudo tmutil deletelocalsnapshots [DATE_STAMP]

# Clear user application cache directory
rm -rf ~/Library/Caches/*

# Purge local npm package cache
rm -rf ~/.npm

4 // PERMISSIONS SOP

Enforce file system boundaries. By default, new shell files are not executable. You must manually grant permission vectors to the owner.

// ACCESS_CONTROLS_SHELL 
# Make a local script executable for the owner only
chmod +x run automation.sh


# Restrict reading and writing of a sensitive database to owner only
chmod 600 production vault.db


# Verify permissions list of current path
ls -la
